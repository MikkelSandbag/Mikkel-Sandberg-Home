My Work

Experts Exchange (Website)

While working as a front end developer at Experts Exchange, I had the opportunity to work on many projects and help create the look and feel of the site. Some of my work that I am most proud of are the user profile redesign, the Live product redesign, and the creation of the Headlines feed. You can see more details on each below.

Tools Used

  • HTML/CSS/JS
  • Photoshop
  • Illustrator

Profile Page Redesign

This is the new profile page that my team built. Our goal was to have the first section one sees on the page be a "business card" that would show information pertaining to the user's areas of expertise and skills. Further down in the overview section, the user can give information on their background (work history, education, and biography) to serve as a résumé. There are four other sections that asynchronously load in, which allow one to see a user's recent contributions on the site, certifications earned, ratings from Live or Gigs interactions, and endorsements that the user has given or received. On the right side, one can see the overall "level" of that user, see if the user has a "verified" account, and one can share the profile on social media. The profile owner also has a "preview" button. My role in this project was to create the HTML, CSS, and JS for more advanced user interactions than what pure HTML and CSS can provide.

Live Product Redesign

The Live product allows clients to connect with consultants in a one-on-one conversation to get help with coding problems. The problem we were facing was that clients didn't know how to get started with the product. We used to show a list of all consultants who were online and allow clients to connect with anyone at random, so the requests were not a good fit a lot of the time and both sides got frustrated. So, we streamlined the whole process and made a help request form front and center for the client, as well as gave consultants the ability to set what topics they are willing to help in so that they would only show up in suggestion lists for certain topics. My role in this product was to mock up the initial designs in Photoshop for the help request page that lists the consultants available for a given request, and then to code all of the CSS to pull it off. I also provided some HTML structure when needed, but we were mostly just modifying a lot of components that already existed.

Live Landing Page

This was the logged out view for the Live product. The purpose of this page was to show the viewer why Live is valuable, how it works, and provide some testimonies from actual users — all in a beautifully designed and easy to use interface. I was responsible for taking the mockup provided by our UI team and create all the HTML and CSS needed to bring the page to life.

Headlines Feed

Headlines was intended to be a place for users to see a feed of all the tech news from around the internet. Users had the ability to cater the feed to their needs by hiding specific stories they had already read or were not interested in, hide entire sources, or add new sources by importing RSS feeds. I was responsible for creating the look and feel of the feed, and I worked with my team's UI designer to come up with the final "card" design. Then, I created the HTML and CSS needed to make the cards arrange nicely at any screen size, and adjust to any number of card in the feed.

Friend Quest (Podcast)

Friend Quest is a podcast that I do with my friends where we play tabletop role playing games. We each record our audio separately and then I edit all of the tracks in Audition to get the final cut. Then, I add the intro and outro music, record voiceovers for the intro and outro, and add some basic EQ and noise filtering. Finally, I export the final audio and bring it into Premier to create a video version of the episode as well. The cover artwork was done by Peter Freeby.
